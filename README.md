# 按键, play
---------------------------------------------

* 上	W	up
* 下	S	down
* 左	A	left
* 右	D	right	
* 开始	H	start
* 选择	F	select
* A键	J (比如开枪,打球等) A button
* B键	K (比如跳起,取消等) B button


# 其他, other
---------------------------------------------

按下'x'就可以了,
如果想尝试'调试'功能,按下'h'能看到帮助信息;  

when 'break debug', push botton 'x' on console to return game.
